# Streamaze Discord Bot

The Streamaze Discord bot currently has three main features.

1. Send a timestamp of the current YouTube stream to a Discord channel
2. Allow mods to `/clip` the current YouTube stream
3. Update the OBS Ticker Widget with `/ticker <new text>`

More features will be coming soon, please contact us with your suggestions!

## Connect the Bot

You can invite the bot to your server [using this link](https://discord.com/api/oauth2/authorize?client_id=1058511637662941335&permissions=3136&scope=bot). Make sure the bot has the proper permissions per channel.
