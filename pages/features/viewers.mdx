# Display Viewers

Streamaze can display the number of viewers watching your stream. We display these numbers in the top right of your Dashboard for visibility. To setup this feature, follow the instructions below.

![Viewers](/images/stats/viewers.png)

## Setup Viewer Count

To setup the viewer count, go to the [Settings page](https://streamaze.live/settings) of the Dashboard find the Stats Settings section. Fill in the platform info that you want to see and click Save to save your changes.

![Stats Settings](/images/stats/settings.png)

<Callout>
    Twitch view count will be supported shortly. Currently, we only show YouTube, TikTok and Kick.
</Callout>
