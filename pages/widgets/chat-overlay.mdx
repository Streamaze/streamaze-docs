## Purpose

Displays all of your live chat messages into a single OBS overlay. Useful for VODs.

## Settings

- Add `&theme=default` to the end of the URL to change the theme to light mode.
- Add `&theme=dark` to change the theme to dark mode.
- Add `&theme=big` to change the font size to be larger.

You can make the background transparent by following the instructions [here](/obs-setup#transparent-background).
